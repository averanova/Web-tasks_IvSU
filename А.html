• Алгоритмы — набор инструкций, которые используются для решения задач в программировании.
  • API (Application Programming Interface) — интерфейс прикладного программирования, который позволяет приложениям обмениваться данными и функциями друг с другом.
  • Ajax (Asynchronous JavaScript and XML) — технология, которая позволяет обновлять части веб-страницы без перезагрузки всей страницы.
  • Адрес URL (Uniform Resource Locator) — уникальный адрес веб-ресурса в интернете
  • Адаптивный дизайн — подход к созданию веб-сайтов, при котором сайт автоматически подстраивается под размер экрана устройства пользователя.
