• Домен — уникальная область в интернете, которая идентифицируется с помощью доменного имени.
  • DOM (Document Object Model) — объектная модель документа, которая представляет структуру веб-страницы в виде дерева объектов.
  • Динамический сайт — сайт, который генерирует контент на основе данных из базы данных или других источников.
  • Дебаг (отладка) — процесс поиска и устранения ошибок в программном коде.
